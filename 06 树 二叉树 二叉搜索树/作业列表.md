树节点的定义

```java
public class TreeNode {
	public int val;
  public TreeNode left, right;
  public TreeNode (int val) {
    this.val = val;
    this.left = null;
    this.right = null;
  }
}
```

 前序遍历 pre-order 根左右

中序遍历 in-order 左根右

后序遍历 post-order 左右根



#### 二叉搜索树binary-search tree

也称有序二叉树（ordered binary tree），排序二叉树（sorted binary tree）

空树也是二叉搜索树

1.左**子树**==所有节点==小雨根节点

2.右**子树**==所有节点==大雨根节点

中序遍历： 生序排列
